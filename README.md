# Oculus-Qwest-3-Stereo-3D-Camera
# 📷 Oculus Quest 3 Stereo Camera Script

## 🇷🇺 Русская версия

### 📌 Описание

Скрипт для **Oculus Quest 3** (на других моделях не проверялся), позволяющий включить запись и вывод камеры VR-шлема в **стерео-режиме**.

> ⚠️ Использование на других устройствах не гарантируется.

---

### 🎥 Источник информации

Информация взята из видео:

https://www.youtube.com/watch?v=Odx5bNwdvOA&t=2s

---

### ⚙️ Установка

1. Скачайте **Android SDK Platform-Tools**:
   https://developer.android.com/tools/releases/platform-tools?hl=ru

2. Выберите:

   > **Скачать SDK Platform-Tools для Windows**

3. Распакуйте архив в удобную папку:

   ```
   C:\platform-tools
   ```

   ⚠️ Желательно, чтобы путь:

   * не содержал кириллицы
   * не содержал пробелов

4. Поместите скрипт в ту же папку.

---

### 🔌 Подключение

* Включите **режим разработчика** на шлеме
* Подключите шлем к ПК через USB
* При запуске подтвердите:

  > **USB Debugging (отладка по USB)** внутри шлема

---

### 🎬 Просмотр стерео-видео

1. Установите приложение:
   **DEOVR Video Streaming** (из магазина Oculus)

2. Запустите приложение

3. Посмотрите вниз — откроется меню

4. Нажмите:

   * 📁 **Local Files**

5. Выберите видео

---

### 🧠 Настройка отображения

Стерео-видео выглядит как:

> два изображения рядом (Side-by-Side)

В плеере:

* В левом нижнем углу:

  * Режим: **Flat**
  * Формат: **Side by Side (вместо Mono)**

---

### 🎮 Управление

* Левый джойстик:

  * Приближение / отдаление экрана

---

## 🇬🇧 English Version

### 📌 Description

A script for **Oculus Quest 3** (not tested on other devices) that enables **stereo camera recording and output** from the VR headset.

> ⚠️ Compatibility with other models is not guaranteed.

---

### 🎥 Source

Information is based on this video:

https://www.youtube.com/watch?v=Odx5bNwdvOA&t=2s

---

### ⚙️ Installation

1. Download **Android SDK Platform-Tools**:
   https://developer.android.com/tools/releases/platform-tools

2. Choose:

   > **Download SDK Platform-Tools for Windows**

3. Extract the archive to a directory, for example:

   ```
   C:\platform-tools
   ```

   ⚠️ Recommended:

   * No spaces in path
   * No non-Latin characters

4. Place the script into the same folder.

---

### 🔌 Connection

* Enable **Developer Mode** on your headset
* Connect the headset via USB
* Confirm:

  > **USB Debugging** inside the headset

---

### 🎬 Watching Stereo Video

1. Install:
   **DEOVR Video Streaming** (from Oculus Store)

2. Launch the app

3. Look down to open the menu

4. Click:

   * 📁 **Local Files**

5. Select your video

---

### 🧠 Display Settings

Stereo videos appear as:

> two images side-by-side

In the player:

* Bottom-left corner:

  * Mode: **Flat**
  * Format: **Side by Side (instead of Mono)**

---

### 🎮 Controls

* Left joystick:

  * Zoom in / zoom out the screen
